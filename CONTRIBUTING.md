## How to Contribute

We welcome technical contributions that improve the security and functionality of the system:

1.  **Issues**: Report bugs or suggest features via the Issue Tracker.
2.  **Pull Requests**: 
    *   Fork the repository.
    *   Create a feature branch.
    *   Submit a PR with a detailed description of your changes.
    *   **Note**: Submitting a PR constitutes agreement to the licensing terms above.

## Code of Conduct

*   Act with integrity.
*   Respect the proprietary nature of this project.
*   Focus on enhancing security and reliability for the community of analysts using this tool.
