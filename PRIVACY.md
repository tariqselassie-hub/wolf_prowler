# Privacy Policy

**Effective Date**: 2025-01-14  
**Last Updated**: 2025-01-14

## Overview

Wolf Prowler is designed with a **zero-knowledge architecture**. We do not collect, store, or have access to any user data processed by the software.

## Zero-Knowledge Architecture

### What This Means

- **No Central Servers**: All operations are peer-to-peer
- **No Data Collection**: We do not collect any user information
- **No Telemetry**: No usage statistics or analytics
- **No Logging**: No centralized logs of user activities
- **End-to-End Encryption**: All data remains encrypted between peers

### User Control

Users have complete control over:
- Their own data
- Encryption keys
- Network connections
- Audit logs (stored locally)

## Data Handling

### Data We Do NOT Collect

- Personal information
- IP addresses
- Network traffic content
- Command history
- User credentials
- Usage patterns
- Analytics or telemetry

### Data Users Control

Users are responsible for:
- **Local Audit Logs**: Encrypted logs stored on user's system
- **Cryptographic Keys**: Generated and stored locally
- **Configuration Files**: Stored on user's system
- **Network Metadata**: Peer discovery information (ephemeral)

## Third-Party Services

Wolf Prowler does not integrate with any third-party services that collect user data.

### Optional Integrations

If users choose to integrate with external services (e.g., cloud providers, monitoring tools), those integrations are:
- Entirely optional
- Configured by the user
- Subject to the third-party's privacy policy
- Not controlled or monitored by us

## Cryptography and Privacy

### Encryption

All sensitive data is encrypted using:
- **Post-Quantum Cryptography**: ML-DSA-87, ML-KEM-1024
- **Symmetric Encryption**: AES-256-GCM
- **Zero-Knowledge Proofs**: Where applicable

### PII Protection

The software includes:
- **PII Detection**: Automatic detection of personally identifiable information
- **PII Blocking**: Commands containing PII are rejected
- **Encrypted Audit Logs**: Audit trails are encrypted and stored locally

## User Responsibilities

Users are responsible for:

1. **Secure Key Management**: Protecting their private keys
2. **Access Control**: Configuring appropriate access policies
3. **Audit Log Security**: Protecting local audit logs
4. **Compliance**: Ensuring their use complies with applicable laws

## Data Retention

Since we do not collect any data, there is no data retention policy. All data remains under user control.

## Geographic Restrictions

### US-Only Distribution

This software is intended for use within the United States only. We do not:
- Market to international users
- Provide support for international deployments
- Collect data from international users

This software contains cryptographic technology and is subject to US export control regulations. See EXPORT_CONTROL.md for details.

## Children's Privacy

Wolf Prowler is not intended for use by individuals under 18 years of age. We do not knowingly collect information from children.
