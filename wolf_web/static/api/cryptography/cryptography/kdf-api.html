Parameters

algorithm (string, optional) - argon2, pbkdf2, scrypt

Response Format

{
  "success": true,
  "derived_key": "generated_key_here",
  "algorithm": "argon2"
}

Example

curl -X POST http://localhost:8080/api/kdf \
  -H "Content-Type: application/json" \
  -d '{"password": "secret", "salt": "salt", "algorithm": "argon2"}'
