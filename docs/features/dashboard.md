# Omega Dashboard

The central management interface for the Wolf Prowler network.

## Features

- **Real-time Monitoring**: WebSocket-based live updates of node status and network activity.
- **Network Map**: Interactive visualization of node topology and geographic distribution.
- **Security Alerts**: Instant notification and drill-down into security incidents.
- **Metric Dashboard**: High-level overview of CPU, memory, and network performance across the swarm.

## Technology Stack

- **Backend**: Axum-powered high-performance web server.
- **Frontend**: Vanilla CSS and Javascript for maximum speed and zero dependencies.
- **Templating**: Askama type-safe templates for Rust.
