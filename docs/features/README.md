# Wolf Prowler Features

Comprehensive index of the features and capabilities of the Wolf Prowler ecosystem.

## Core Systems

- **[Cryptography](cryptography.md)**: Pure Rust cryptographic primitives, advanced encryption, and key management.
- **[Networking](networking.md)**: P2P communication, Raft consensus, and [HyperPulse](HYPERPULSE.md) (QUIC) transport. See also: [Alpha Election](alpha_election.md).
- **[AI & Machine Learning](ai_ml.md)**: On-node analytics, threat detection models, and ONNX integration.

## Security & Operations

- **[Threat Intelligence](threat_intelligence.md)**: SIEM processing, real-time event correlation, and external threat feeds.
- **[Cloud & Container Security](cloud_container.md)**: Multi-cloud integrations (AWS, Azure, GCP) and Kubernetes runtime security.
- **[Compliance & Auditing](compliance.md)**: Automated reporting, X509 certificate auditing, and compliance tracking.
- **[Prestige System](PRESTIGE_SYSTEM.md)**: Node reputation and trust decay mechanics.

## Interfaces

- **[Omega Dashboard](dashboard.md)**: Modern web-based visualization and [Omega management interface](OMEGA_DASHBOARD.md).
- **[Wolf Control](tui_control.md)**: High-performance terminal user interface for node management.
