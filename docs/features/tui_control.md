# Wolf Control (TUI)

The ultimate terminal interface for node operators.

## Features

- **Fast Management**: Low-latency interaction with local and remote nodes.
- **Diagnostic Views**: Detailed live metrics and log tailing in the terminal.
- **Security Controls**: Direct access to firewall rules and security configuration.
- **Authenticated Access**: Secure terminal-to-node communication.

## Technology Stack

- **UI Framework**: `ratatui` for high-performance terminal rendering.
- **Backend**: Direct integration with `wolf_net` and `wolfsec` types.
- **Input**: `crossterm` for cross-platform terminal control.
